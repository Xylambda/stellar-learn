# stellar-learn

Stellar classification in Python.

## Installation

```bash
pip install git+https://github.com/Xylambda/stellar-learn
```

### Developers

```bash
git clone https://github.com/Xylambda/stellar-learn
pip install -e ./stellar-learn
```
